# 🤖 Simple Python Chatbot

A beginner-friendly chatbot built using Python. The chatbot responds to a few predefined questions and continues the conversation until the user types **"bye"**.

---

## 📌 Features

- Greets the user at startup
- Responds to:
  - Hello
  - How are you
  - What is your name
- Ends the conversation when the user types **bye**
- Displays a default message for unknown inputs
- Simple and easy to understand for Python beginners

---

## 🛠️ Technologies Used

- Python 3.x
- Console/Terminal
